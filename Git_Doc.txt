Git Commands:
1. Initilate git:
    git init
2. ls:list of files
3.git status:
      to check the files or tracked or untracked.
4. git add <filename>: 
      to add a file into tracked area
5.git config user.name: 
       to check the user name
6. git config user.name "username":
       to configure username
7.git config user.email <email of user>
      to configur email id
8. git remote:
     to check any remotes avaliable or not
9.git remote add <remoteName> path or url:
      to add remote
10. git commit -m "commit message"
       to write a commit message
11. push the data:
    git push remotename branchname
